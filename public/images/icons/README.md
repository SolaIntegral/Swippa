# Material Icons ダウンロードガイド

このフォルダには、Google Material Iconsからダウンロードしたアイコン画像を配置します。

## ダウンロード方法

1. [Google Material Icons](https://fonts.google.com/icons)にアクセス
2. 各アイコン名で検索
3. サイズ: **24px** または **48px**（用途に応じて）
4. 色: **#1f1f1f** (黒) または **#3498db** (ブルー) または **#ffffff** (白)
5. 形式: **PNG** または **SVG**（推奨: SVG）

## 必要なアイコン一覧

### ナビゲーションアイコン
- `home` または `dashboard` - メイン画面（24px, 黒 #1f1f1f）
- `calendar_today` - スケジュール画面（24px, 黒 #1f1f1f）
- `person` または `account_circle` - プロフィール画面（24px, 黒 #1f1f1f）

### アクションアイコン
- `add` - FABボタン（24px, 白 #ffffff）
- `close` - モーダル閉じるボタン（24px, 黒 #1f1f1f）
- `check_circle` - 完了アイコン（24px, 緑 #2ecc71）
- `trending_up` - 成長アイコン（24px, オレンジ #f39c12）
- `fitness_center` - 突破アイコン（24px, オレンジ #f39c12）

### モード切替アイコン
- `power_settings_new` - ON/OFFスイッチ（24px, ブルー #3498db）
- `sleep` または `hotel` - OFFモード（48px, 青 #3498db）
- `work` または `fitness_center` - ONモード（48px, オレンジ #f39c12）

### その他
- `star` - オンボーディング（48px, 黄色 #f1c40f）
- `arrow_back` - 戻るボタン（24px, 黒 #1f1f1f）
- `arrow_forward` - 次へボタン（24px, 黒 #1f1f1f）

## ファイル命名規則

```
{アイコン名}_{サイズ}_{色}.{拡張子}
例: add_24_white.svg, calendar_today_24_black.svg
```

または、よりシンプルに：
```
{アイコン名}.{拡張子}
例: add.svg, calendar_today.svg
```

## 色の定義

- **黒**: #1f1f1f または #333333
- **白**: #ffffff
- **ブルー**: #3498db（プライマリカラー）
- **緑**: #2ecc71（成功）
- **オレンジ**: #f39c12（警告・強調）
- **黄色**: #f1c40f（ハイライト）
